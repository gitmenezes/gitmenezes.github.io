Como Automatizar a sua Rotina

Siga estes três passos simples para criar a sua primeira macro.

1

Iniciar Gravação

Clique no botão principal. A janela irá minimizar-se para libertar o seu ecrã.

2

Executar Tarefas

Realize a tarefa desejada (cliques, digitação). O sistema grava o tempo real entre as ações.

3

Tecla Mestra ESC

Pressione ESC para finalizar. A janela reaparecerá com a gravação pronta para ser salva.

// Configurações de Reprodução

repeticoes: 100

velocidade: 2.0 // 2x mais rápido

intervalo: 5.0 // Aguarda 5s entre ciclos

Pausas Inteligentes

O sistema respeita o tempo exato entre as ações. Se fizer uma pausa de 3 segundos durante a gravação, a reprodução repetirá exatamente esse tempo.
